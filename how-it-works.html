Common Questions

Can kids access YouTube directly?

Viddy only shows channels you've approved. However, Viddy is a web app ‚Äî it doesn't replace parental controls at the device or browser level. For full protection, use it alongside your device's built-in screen time settings.

Does it work on all devices?

Yes ‚Äî Viddy works on any device with a modern browser (iPad, Android tablet, phone, computer). You can install it to the home screen as an app on both iOS and Android.

How do I install it to the home screen?

On iPhone/iPad: tap the Share button in Safari then choose Add to Home Screen. On Android: tap the browser menu and choose Install app or Add to Home Screen.

Can I have multiple parent accounts?

Each account has its own set of approved channels and controls. Whichever parent account last signed in to the dashboard is the one the kids' home screen shows channels for.
